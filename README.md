# Ejercicio-5-html

Este ejercicio practica la implementación de enlaces externos utilizando rutas absolutas para dirigir al usuario a diferentes niveles de un sitio web.

## Especificaciones de la Entrega
El archivo contiene dos tipos de navegación externa hacia el diario **El País**:

1.  **Enlace de Dominio:** Un hipervínculo que apunta a la página raíz del periódico, mostrando únicamente el nombre del dominio (`elpais.com`) como texto visible.
2.  **Enlace a Página Determinada:** Un segundo hipervínculo que redirige específicamente a la sección de **Tecnología**, demostrando el uso de rutas internas dentro de un servidor externo.

## Etiquetas Aplicadas
*   `<a>`: Con el atributo `href` configurado con protocolos seguros (`https://`).
*   `<p>`: Para organizar visualmente los enlaces en bloques separados.

## Objetivo
Diferenciar entre una URL de inicio y una URL de sección específica, asegurando que el destino del hipervínculo coincida con la descripción del texto.
